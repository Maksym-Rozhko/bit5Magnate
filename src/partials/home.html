User Level System

As soon as the first transaction appears in the blockchain, the site will start a 120 second countdown timer.
If no new BIT5 purchase transaction for 50 USDT appears in the blockchain before the timer reaches 0, the purse that bought last becomes the winner and receives 1 BTC. If there was another purchase in the network before the timer zeroed out, the system will start a new countdown from the moment of that transaction.

The action duration is defined as 1 month from the start at 00:00:00 on 01.07.2022 and after that time if the timer has not stopped earlier the timer will be stopped forcibly at 00:00:00 on 01.08.2022. The last recorded online BIT5 transaction of 50 USDT becomes a winner and receives 1 BTC.
